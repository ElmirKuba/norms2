# ADR-0049: Продуктовые принципы «Акцента» — Внешняя память и Самокомандование

- Статус: принято (2026-06-15, фаза 2 / вычитка доки); дополнено 2026-06-17 (Принцип 3 + Миссия — философская сессия → продукт)
- Контекст: при ревизии «Акцента» сформулированы два ядровых принципа, которые задают
  *смысл* раздела (не механику). Записаны в `README §6` раздела; здесь — каноничное
  «почему» + опора на признанные практики. Дополняют принципы README/ADR-0027.

## Принцип 1 — Внешняя память (разгрузка головы)

Голова — для мышления, не для хранения (**GTD**, David Allen: «mind for ideas, not
for holding them»; **Building a Second Brain**, T. Forte; когнитивная разгрузка;
**эффект Зейгарник** — открытые петли тревожат). Все намерения/планы/контекст живут
в системе — **единый источник истины** (как store на фронте), не в уме. Система не
вываливает всё, а **отдаёт нужный срез в нужный момент** (just-in-time / progressive
disclosure): БД = store, `Recommender`/дашборд = «селектор» над ним.

## Принцип 2 — Самокомандование

Человек в моменте (уставший, в тумане) плохо решает «что делать»; он же в ясном
состоянии — хороший командир. Поэтому: **план задаёшь заранее → система в моменте
директивно ведёт по нему** («Сейчас: …», 1 действие в один тап), масштаб под состояние,
**но никогда не виня за пропуск**. Команда легитимна, потому что **своя** (commitment
device; автономия по теории самодетерминации, SDT) — не ломает мотивацию, а держит.
Самокомандование — активное проявление Внешней памяти.

## Принцип 3 — Тренажёр отказа (вычитание, не добавление) (доп. 2026-06-17)

**Свобода = (есть из чего выбирать) + власть осознанно отказываться и фильтровать ради
важного** (уточнение Elmir 2026-06-17: не «или», а ОБА; ср. М. Мэнсон — «выбирай, на что
не плевать»). Обладание вариантами — сырьё; свобода — направленный фильтр над ним: чистый
отказ без вариантов = бедность в обёртке, чистые варианты без фильтра = утопание. «Акцент»
тренирует именно мышцу фильтра/отказа. Само имя — *ударение*: выделить
важное можно только **приглушив остальное**. Отстрой от конкурентов: типовые трекеры —
«тренажёры ДОБАВЛЕНИЯ» (ещё привычка/задача/стрик → доливают в множество, где человек и
тонет); «Акцент» — **тренажёр ВЫЧИТАНИЯ**. Слоган: **«Акцент учит отказываться от лишнего
ради важного»**. Думскролл — провал этой свободы (бесконечное множество захватывает);
микро-победа вместо ленты — один повтор отказа. Не аскеза ради аскезы, а **направленный**
отказ (пример: богатый, который может купить яхту и отказывается — свободен; бедный, что о
ней мечтает — нет).

**Фильтр на КАЖДУЮ фичу/метрику/уведомление:** «тренирует отказ ради важного — или просто
доливает в множество (шум)?» Доливает → отклоняем, даже если «полезно». **Сдержанность —
это бренд** (страховка от раздувания продукта на 11 фаз).

## Миссия — расширить «бутылочное горло» (доп. 2026-06-17)

Ошибка выжившего работает **на всех уровнях**: «один из ста» прорывается и у бедных
(остальных сожрала нехватка), и у предпринимателей (убили/отжали/посадили/не хватило
запала — о них просто не узнали). Двигатель прорыва — **не лишение** (это фильтр с
потерями), а **запал + петля «усилие→результат» + смысл**. Миссия «Акцента» — поставить
эту петлю + внешнюю память + самокомандование **массово и безопасно**, чтобы **расширить
горло**: прорывался не 1 из 100. Конечная цель — растить людей, которые **ценят,
приумножают и передают дальше** — и главное, передают дальше **сами эти три способности**
(само-реплицирующаяся культура, не разовый успех; ср. «идентичность, а не правила»).

## Решение

Принять все принципы как **ядровые принципы раздела** (П1/П2 — README §6; П3 «тренажёр
отказа» + Миссия — слоган вынесен в README §1). Guardrail: директивный тон ≠ вина — тон
«интерес, не страх» обязателен (защита от Skinner-box / страха).

## Последствия

- Ориентир для `Recommender` (подфаза 2.7) — отдаёт директивную «команду сейчас».
- Ориентир для **дашборда** (2.10) — «Сейчас/рекомендации» = герой экрана, не пассивный
  список из 20 задач.
- Согласуется с позиционированием (свободно/бесплатно/open-source, без paywall) и
  с anti-burnout/non-Skinner курсом (memory `accent-sustainable-achievement-design`).
- **Геймификация (2.8):** награды должны ощущаться **заработанными**, не дармовыми — это
  петля «усилие→результат» на уровне механики (anti-Skinner). Работает и на себе (user
  zero), не только «на детях».
- **Фильтр П3** применяется к каждой будущей фиче (вычитание vs доливание в множество).
- **Кандидат-фича (2.12+):** линза «вспомню ли я об этом через неделю?» — критерий
  важного при оценке цели/задачи/действия (отделяет смысл от шума).
