# ADR-0022: Концепция и философия (A1–A6)

- **Статус:** accepted
- **Дата:** 2026-05-30
- **Решает:** Elmir (пункты A1–A6)
- **Контекст-теги:** [product] [concept]

## Решение

- **A1 — тон:** ламповое/мемное, дружелюбный закрытый клуб. Псевдонимы поощряются («БабаЯга», «Нагибатор 2000»). Копирайт — неформальный, с юмором, но уважительный. Это база для `ui-ux.md`.
- **A2 — закрытость:** **invite-only всегда** в проде (`FREE_REGISTRATION=false`). Открытие регистрации — точечно/вручную при нужде. `FREE_REGISTRATION=true` — только в dev.
- **A3 — языки:** фаза 1 — **только русский**; i18n-структура заложена, другие языки — позже.
- **A4 — монетизация:** **некоммерческий**. Реклама и продажа ПДн — **никогда**. Будущая устойчивость (не фаза 1): добровольные донаты **строго на инфраструктуру**, прозрачный открытый счёт, **без** гейта фич/привилегий/данных.
- **A5 — админ-роль:** **отложена** (не фаза 1). Первый конкретный повод — override квоты инвайтов ([ADR-0007](./0007-invite-quota-counter.md)). Модель данных уже допускает (правки на уровне БД), UI/роль — позже.
- **A6 — профиль:** единый на все разделы (общий ЛК), как в [Technical-assignment.md](../../Technical-assignment.md). Подтверждено.

## Последствия
- `ui-ux.md`: тон копирайта — ламповый/мемный; русский; никаких намёков на платность.
- Прод по умолчанию `FREE_REGISTRATION=false`.
- ToS/Privacy — некоммерческий, без рекламы/ПДн.
- Будущий ADR по донатам — когда дойдёт (за пределами фазы 1).
