# Раздел «НоваСкил» — Фаза 3

> Подраздел Нормисов: платформа обучающих курсов (курс → глава → урок) для **аутентифицированных** пользователей. Общий ЛК и БД с остальными разделами. Решения — [ADR-0029](../../decisions/0029-novaskil-phase3-core.md). История переваривания — `../../phase3-analysis-map.md`.

## 1. Что это
Раздел онлайн-обучения «для своих»: админы создают курсы, аутентифицированные пользователи проходят их и копят прогресс. Часть площадки Нормисы наравне с ЛК / Акцент / Мессенджер.

## 2. Привязка к фундаменту
- Аккаунты, сессии, id `uuidv7___unixmillis`, 5-слойный Nest+Drizzle ([ADR-0030](../../decisions/0030-stack-revision-drizzle-5layer-npm.md)), `/api/v1`, SCSS+MatDialog — из фазы 1. Свой ЛК/email — НЕ заводим.
- **Роли:** платформенные `roles` + `account_roles` (N:M), вводятся этим разделом ([domain-model](./domain-model.md)).
- **Контент** — JSON-файлы + медиа на диске (`content/`), раздаётся статикой ([content-format](./content-format.md)).

## 3. Состав
| Часть | Где | Кратко |
|---|---|---|
| Роли | БД (Drizzle) | `roles`, `account_roles` — кто админ |
| Контент | диск (JSON) | Курс → Глава → Урок, `index.json`, медиа |
| Прогресс / enrollment | БД (Drizzle) | отметка пройденных уроков на `account` |
| Редактор | фронт (admin) | TipTap (md/html, визуал+исходник) |

Подробно: [domain-model](./domain-model.md) (БД), [content-format](./content-format.md) (диск), [api-contracts](./api-contracts.md), [ui-ux](./ui-ux.md).

## 4. Роли и доступ ([ADR-0029](../../decisions/0029-novaskil-phase3-core.md) N2)
- `admin` — CRUD курсов/глав/уроков, управление пользователями/ролями.
- `user` (по умолчанию любой аутентифицированный) — чтение курсов + прогресс.
- Один аккаунт может иметь несколько ролей (через `account_roles`).

## 5. Принципы
- ПДн не собираем (общий режим [ADR-0001](../../decisions/0001-data-minimization-no-pii.md)).
- Тон «интерес не страх», accessibility-first, скелетоны при загрузке.
- Прогресс — серверная истина по уроку; агрегаты курса/главы вычисляются.

## 6. MVP и волны ([ADR-0029](../../decisions/0029-novaskil-phase3-core.md) N7)
- **MVP:** роли, CRUD контента (курс/глава/урок) на диске, все типы контента (md/video/link/html), прогресс по уроку, редактор TipTap, медиа на диске, прохождение курсов.
- **Волны:** геймификация обучения (N5 — пока нет), связь курс↔Акцент «Своё обучение» (N6 — задел заложен), сертификаты, поиск/фильтры по тегам.
