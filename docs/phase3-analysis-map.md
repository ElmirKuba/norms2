## 0. Видение Фазы 3 (захвачено своими словами — не потеряется)

«НоваСкил» — **протокол передачи мастерства** внутри Нормисов: раздел, где один человек (`Senior`, кто умеет) ведёт другого (`Normis`, кто учится) через освоение навыка по микрошагам. Не курсы — а структурированная передача навыка «для своих».

Три кита: **микрошаги** (навык дроблён на атомы, которые невозможно не осилить — рифма с лесенкой Акцента), **интервальные повторения** (spaced repetition против забывания), **эмоциональная безопасность** (ошибка = данные, не позор — рифма с антихрупкостью).

Главный принцип: «Ты не обязан быть умным. Ты обязан быть последовательным.»

**Роли — контекст, не статус:** один и тот же человек Senior в одном навыке и Normis в другом. Ложится на инвайт-дерево/круг доверия Нормисов.

North star: дать «Васе» возможность реально освоить навык в системе, учитывающей, как работает мозг, без стыда и перегруза.

## 2. Леджер идей (из протокола, переформулировано)

**Сущности-каркас:**
- **Skill** (навык) = дерево узлов. Принадлежит автору-Senior.
- **SkillNode** (узел): title, description, `prerequisites` (узлы-предусловия), atoms[], `artifactRequired` (нужен ли артефакт для зачёта).
- **Atom** (атом знания): минимум 5–15 мин, содержит теорию(кратко)+пример+micro-задание. Правило: не осваивается за присест → дроби дальше. ← прямая рифма с «минимальной победой».
- **Enrollment / Progress**: Normis проходит дерево; на каждого — персональная очередь повторений и статус узлов/атомов.
- **Artifact** (доказательство освоения): сдаётся для зачёта узла; проверка авто или Senior.
- **ReviewItem** (очередь повторений): на атом, по SM-2.
- **Mentorship**: связь Senior↔Normis (кто кого обучает по навыку).

**Роли:** Senior (создаёт/версионирует дерево, проверяет/делегирует, даёт обратную связь без токсичности) · Normis (проходит, сдаёт артефакты, получает XP/уровни/бейджи). Роль = контекст навыка, не статус личности.

**Сессия обучения (15–45 мин):** разогрев (1–2 вопроса повторения) → новый атом (теория+пример) → практика (micro-задание) → проверка (авто-тест/чеклист/Senior) → рефлексия (что понял/нет, фиксируется). **Правило: 1 атом за сессию.**

**Anti-forgetting (SM-2 упрощённый):** интервалы 1→3→7→16→35 дней; `easiness` старт 2.5; верно → `interval *= easiness`; неверно → `interval = max(1, interval*0.5)`, `easiness -= 0.2` (мин 1.3). **Откат на предыдущий интервал, НЕ на ноль** (берегём психику). Повторение = 1–2 быстрых вопроса.

**Геймификация:** XP атом 10 / узел (атомы+артефакт) 50 / навык 500; серии (дни с ≥1 сессией); бейджи (Первый атом, Первое дерево, 30 дней, Ментор); уровни Normis по навыку: Новичок→Ученик→Подмастерье→Мастер. Анти-абуз: XP только за проверенные атомы.

**Защита от перегруза:** ≤3 новых атома/день (повторения не в счёт); 3 ошибки подряд → сессия мягко завершается; режим «только повторения» (плохой день); дедлайнов нет по умолчанию (свой темп).

**Эмоциональная поддержка:** ошибка → поддержка не критика; Senior шлёт ободрение; нет публичных рейтингов (сравнение только с собой вчерашним); микро-победы отмечаются.

**Техтребования платформы:** хранение/версионирование деревьев; персональная очередь повторений; проверка артефактов (авто/ручная); in-app уведомления о готовых к повторению; прогресс-дашборд (дерево с подсветкой освоенного); связь Senior↔Normis.
