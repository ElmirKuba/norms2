## A. Создание + поля
- [ ] **A1** название + категория + длительность + цена энергии → создаётся; уходит **в конец** списка.
- [ ] **A2 категории** — все 11 (телесная/умственная/эмоциональная/социальная/сенсорная/бытовая/цифровая/отдых/духовная/природа/границы) выбираемы.
- [ ] **A3 энергия** — допустимо 1..3; вне диапазона → ошибка (понятная, форма не закрывается).
- [ ] **A4 длительность** — допустимо 0..300 сек; вне → ошибка.
- [ ] **A5 пустое название / >120 символов** → ошибка.
- [ ] **A6 «эффект»/состояния** (если есть в форме) — необязательны; сохраняются.

## B. Выполнение (дневной лимит, реактивно без F5)

- [ ] **B2 дневная идемпотентность** — повторный тап в тот же день → **без дублей** (`completedToday` остаётся, лог один). Проверено бэк-смоуком (2 запроса → 1 лог).
- [ ] **B3 следующий день** — выполненная вчера снова доступна сегодня (новый день — новый лог).
- [ ] **B4 adoption** — выполнение **примера** (is_starter) «присваивает» его себе (бейдж «пример» исчезает).
